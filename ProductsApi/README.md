# 📦 Products API (Java)

Microservicio encargado de la gestión de productos disponibles en la plataforma.

## 🚀 Ejecución
```bash
mvn spring-boot:run
```

📍 Puerto: 5002

## Endpoints

### Obtener productos
```bash
GET http://localhost:5002/api/products
```

### Health Check
```bash
GET http://localhost:5002/api/products/healthcheck
```
